# Caliper Flow 버전 관리 로그

## [v1.3.0] - 2026-01-21
### 추가 및 개선
- 휴지통 기능 강화: '전체 비우기' 기능 추가
- 안전장치 추가: 복원, 영구 삭제, 전체 비우기 시 확인 모달(2단계 인증) 적용
- 설정 화면 개선: 직원 관리 및 휴지통 섹션 접기/펴기(Collapsible) 기능 적용
- 공정 관리 개선: 개별 작업(LH/RH) 방향 표시 명확화
- 공정 관리 개선: 스테이지별 '전체 선택' 기능 추가
- 작업 요청 개선: 작성자(담당자) 선택 필드 필수화

### 버그 수정
- 대시보드에서 '생산완료' 클릭 시 리스트가 표시되지 않던 필터링 오류 수정
- 모달 내 드롭다운(Select) 글자색 시인성 개선 (다크 모드 대비 최적화)
- 대시보드 카운팅 로직 수정 (개별 품목이 아닌 카드/세트 단위로 집계)
